The final model is a Keras sequential model with 3 32-node layers and can be found in the folder on this repo:
"final_nn_model_user_revenue"

There is one dropout layer to zero out 20% of the nodes in the first layer.  
I trained the model with a batch size of 500 and an early stopping criteria to prevent overfitting.

The final metrics for the training set are below
-- set -- 	-- RMSE -- 
training	1.629
validation 	1.639
test		1.658


There are 290 input featuresand they are listed below in order.
Input features need to be transformed to mean of 0 and deviation of 1 with scaler.
The standard scaler is saved using 'pickle' in this repo:
"StandardScaler.pkl"
